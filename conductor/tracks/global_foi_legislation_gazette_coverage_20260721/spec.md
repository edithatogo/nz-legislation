# Global FOI legislation and Gazette coverage

Issue: [legislation #105](https://github.com/edithatogo/legislation/issues/105). Programme: [foi-o #81](https://github.com/edithatogo/foi-o/issues/81).

Incrementally cover every target in the FOI-O roadmap. For each jurisdiction, first verify whether this repository can access the official legislation through an existing provider. Add an official API/provider where available; otherwise create a governed manual-acquisition record with source identity, effective date, rights, retrieval date and hashes. Separately identify the Gazette or equivalent and preserve commencement, subordinate-instrument, implementation and regulator material relevant to FOI process/rules.

No source observation is a legal conclusion. No fabricated legal data, cross-profile import, stable-support claim, publication or release is authorized.

## Acceptance

- Every roadmap target has an independently reviewable legislation and Gazette/equivalent status record.
- Added providers have positive and negative fixtures, provenance and schema checks; manual sources are immutable and reproducible.
- Outputs cross-reference downstream extraction and modelling work and retain temporal scope and unresolved gaps.
