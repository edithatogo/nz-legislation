## Active Tracks

### [ ] Track: P1 - Legislative Volatility

**Link:** [./tracks/p1-legislative-volatility/](./tracks/p1-legislative-volatility/)

**Status:** ⏳ **READY** (awaiting ethics approval)

**Research Question:** What is the tempo and pattern of legislative change in NZ health law across reform cycles, and does reform frequency outpace implementation capacity?

**Differentiation:** Focuses on legislative change tempo and implementation gaps, distinct from prior "policy whiplash" framing which emphasized political direction shifts.

**Priority:** 🔴 **HIGH**
**Timeline:** 10-12 weeks
**Key Deliverables:** NZMJ manuscript, supplementary appendix, figures/tables, policy brief

**Robustness Features:**
- Inter-rater reliability (κ > 0.7)
- Sensitivity analysis
- Expert review
- Triangulation
- OSF pre-registration
- Open science (code + data sharing)

**Current Phase:** Awaiting Phase 1 (Framing)

## Quality Standards (All Tracks)

| Standard | Requirement |
|----------|-------------|
| **Inter-rater reliability** | κ > 0.7 for all coding categories |
| **Sensitivity analysis** | Alternative corpus, time periods, weightings |
| **Expert review** | 1-2 health policy experts, 5-10% sample |
| **Triangulation** | Policy documents, media, parliamentary debates |
| **Pre-registration** | OSF before Phase 4 (Analysis) |
| **Open science** | Code + data shared on acceptance |
| **Word count** | Maximum 3,000 words |
| **References** | Maximum 30 references |
| **Abstract** | 200 words, structured (Aims, Methods, Results, Conclusions) |
