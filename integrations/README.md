# Integrations

This directory contains host-specific integration artifacts for ANZ Legislation.

Keep all integrations in this repository.

## Subdirectories

- `mcp/` — MCP registry and generic host configuration
- `claude/` — Claude Desktop, Claude Code, and skill/prompt-pack artifacts
- `codex/` — Codex AGENTS/prompt-pack artifacts
- `github-copilot/` — Copilot instructions and extension plan
- `vscode/` — VS Code/Open VSX extension plan
- `gemini/` — Gemini/OpenAPI/tool-calling artifacts
- `qwen/` — Qwen/ModelScope/Alibaba tool artifacts
- `jetbrains/` — future JetBrains plugin notes
- `generic-hosts/` — Cline/Cursor/Windsurf/Zed/etc. install notes

Do not publish or submit integration artifacts until the submission gates pass.

The target matrix and current official-source evidence are maintained in
[`assistant-marketplace-readiness.md`](./assistant-marketplace-readiness.md).

GitHub Copilot integration work remains local-only until the release/submission
gates pass.
