## Abstract Requirements (Articles)

**Format:** Structured with four headings  
**Word Limit:** Maximum 200 words

- [ ] **Aims** (1-2 sentences)
  - [ ] Research question/objective clearly stated
- [ ] **Methods** (2-4 sentences)
  - [ ] Study design, setting, participants
  - [ ] Key interventions/exposures
  - [ ] Main outcome measures
- [ ] **Results** (3-5 sentences)
  - [ ] Primary findings with data
  - [ ] Confidence intervals/p-values where appropriate
- [ ] **Conclusions** (1-2 sentences)
  - [ ] Main interpretation and implications

## Reference Style: Vancouver

### In-Text Citations

- [ ] Superscript numbers in order of appearance
- [ ] Example format: `Previous research has shown this.¹`
- [ ] Multiple references: `¹,³,⁵` or `¹⁻³` for consecutive

### Reference List

- [ ] All references numbered consecutively
- [ ] Journal titles abbreviated per **Index Medicus** (not PubMed style)
- [ ] DOI included where available
- [ ] **≤4 authors:** List ALL authors
- [ ] **>4 authors:** First 3 authors + "et al."
- [ ] Websites include access date
- [ ] **Total references:** Maximum 30 (for Articles)

### Reference Format Examples

**Journal Article (≤4 authors - list all):**
```
1. Smith AB, Jones CD, Wilson EF. Title of article. Abbrev Journal Name. Year;Volume(Issue):Page. doi:xxxx
```

**Journal Article (>4 authors - first 3 + et al.):**
```
2. Smith AB, Jones CD, Wilson EF, et al. Title of article. Abbrev Journal Name. Year;Volume(Issue):Page. doi:xxxx
```

**Key NZMJ abbreviations:**
- New Zealand Medical Journal → N Z Med J
- New England Journal of Medicine → N Engl J Med
- The Lancet → Lancet
- JAMA → JAMA
- BMJ → BMJ

## Manuscript Structure

### Title Page (Separate File)

- [ ] Title (maximum 150 characters)
- [ ] Running title (maximum 50 characters)
- [ ] All author names and affiliations
- [ ] Corresponding author details
- [ ] Word counts
- [ ] Conflicts of interest statement
- [ ] Funding sources

### Main Manuscript (Blinded)

- [ ] **Introduction:** 400-600 words
  - [ ] Background and context
  - [ ] Research question/aim clearly stated
  - [ ] Significance explained

- [ ] **Methods:** 800-1,200 words
  - [ ] Study design
  - [ ] Setting and participants
  - [ ] Data collection
  - [ ] Statistical analysis
  - [ ] Ethics approval stated

- [ ] **Results:** 1,000-1,500 words
  - [ ] Participant flow
  - [ ] Primary outcomes with data
  - [ ] Secondary outcomes
  - [ ] Confidence intervals/p-values where appropriate

- [ ] **Discussion:** 1,000-1,500 words
  - [ ] Principal findings
  - [ ] Comparison with literature
  - [ ] Strengths and limitations
  - [ ] Implications for policy/practice
  - [ ] Conclusions

- [ ] **Acknowledgements**

- [ ] **References**

## Ethics and Reporting

### Ethics Requirements

- [ ] Ethics approval stated (if human participants)
- [ ] Ethics committee name included
- [ ] Approval reference number included
- [ ] Example: "Ethics approval was obtained from the New Zealand Health and Disability Ethics Committee (reference 2023/123)"

### Consent

- [ ] Informed consent stated (for case reports)
- [ ] Consent for publication (for identifiable images)

### Clinical Trials

- [ ] Trial registered in approved registry
- [ ] Registration number in abstract
- [ ] Registration number in methods

### Reporting Guidelines

- [ ] Appropriate guideline followed:
  - [ ] CONSORT (randomized trials)
  - [ ] STROBE (observational studies)
  - [ ] PRISMA (systematic reviews)
  - [ ] CARE (case reports)
  - [ ] COREQ (qualitative research)
  - [ ] CHEERS (economic evaluations)
- [ ] Completed checklist uploaded as supplementary material
