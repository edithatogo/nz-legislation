# NZ legislation corpus workflow pack

This package contains a workflow-oriented operating system for the five NZ health legislation corpus studies aimed at pre-2026 election policy relevance and NZMJ publication.

## Included
- `docs/` — markdown documentation
- `code/` — editable workflow code and structured templates
- `images/` — workflow images
- `notion_import/` — Notion-import-ready markdown pages
- `linear/` — Linear-ready initiative and issue seeds
- `docx/` — polished Word version
- `figma_links.md` — live Figma / FigJam links
