# @nz-legislation/commonwealth

Commonwealth (Federal) Australian legislation plugin.

## Installation

```bash
nzlegislation plugin install @nz-legislation/commonwealth
```

## Features

- ✅ Search Commonwealth legislation
- ✅ Retrieve act details
- ✅ Get version history
- ✅ Generate citations
- ✅ Health monitoring
- ✅ Automatic caching

## Data Source

- **Website:** https://www.legislation.gov.au
- **Jurisdiction:** Commonwealth of Australia
- **Coverage:** Federal Acts, Regulations, Legislative Instruments
