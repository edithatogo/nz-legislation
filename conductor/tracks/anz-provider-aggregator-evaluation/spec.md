# Spec: Aggregator Provider Evaluation

## Scope

Evaluate whether any cross-jurisdiction aggregator can assist discovery,
indexing, or metadata comparison without replacing official-source validation.

## Requirements

- Keep aggregator work in this repository and out of runtime support until
  provenance, licence, and source attribution are proven.
- Do not use aggregator data to claim official legal-data coverage.
- Record any candidate as unsupported or evaluation-only in provider capability
  and release materials.

## Contracts

- **Runtime:** no runtime route until source attribution, licence, fixture, and
  provider tests pass.
- **Data:** aggregator data must identify original official sources and preserve
  provider-specific provenance.
- **Publication:** no listing may describe aggregator evaluation as stable
  provider support.
