# Spec: Assistant Integration Marketplaces

## Scope

Prepare Claude, Codex, GitHub Copilot, Gemini, and Qwen integration materials
inside this repository without submitting to any external marketplace or
directory.

## Requirements

- Keep assistant integration docs provider-manifest-aware.
- Separate local instructions and prompt packs from any future marketplace
  submission package.
- Reverify each target's current submission requirements before external action.

## Contracts

- **Artifact:** integration docs, prompts, and instructions must match provider
  capability, package names, and MCP behavior.
- **Submission:** no assistant marketplace or directory submission until target
  requirements, snippets, and security/provenance gates pass.
- **Claims:** copy must not imply first-party endorsement or stable AU support.
