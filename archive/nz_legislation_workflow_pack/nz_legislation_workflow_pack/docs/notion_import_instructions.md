# Notion import instructions

1. In Notion, choose **Import**.
2. Select **Markdown & CSV**.
3. Import the `notion_import` folder from this package.
4. Notion will create pages for the programme hub, workflow overview, and the five protocol pages.
5. After import, create databases for:
   - Projects
   - Decisions
   - Corpus log

The package is structured so the imported pages match the research workflow and publication sequence.
