## Workflow

The research programme follows a 7-phase workflow with stage gates:

1. **Framing** (Week 1) - Gate 1: Framing review
2. **Corpus** (Weeks 2-3) - Gate 2: Corpus review
3. **Coding** (Weeks 4-5) - Gate 3: Coding review (IRR κ > 0.7)
4. **Analysis** (Weeks 6-7) - Gate 4: Analysis review + sensitivity
5. **Drafting** (Weeks 8-9) - Gate 5: Drafting review
6. **Revision** (Weeks 10-11) - Gate 6: Revision review
7. **Submission** (Week 12) - Gate 7: Submission complete
