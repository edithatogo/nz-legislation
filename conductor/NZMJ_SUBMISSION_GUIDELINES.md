## Abstract Requirements (Articles)

**Format:** Structured with four headings  
**Word Limit:** Maximum 200 words

1. **Aims** (1-2 sentences)
   - State the research question or objective

2. **Methods** (2-4 sentences)
   - Study design, setting, participants
   - Key interventions or exposures
   - Main outcome measures

3. **Results** (3-5 sentences)
   - Primary findings with data
   - Include confidence intervals/p-values where appropriate

4. **Conclusions** (1-2 sentences)
   - Main interpretation and implications

## Reference Style: Vancouver

### In-Text Citations
- Superscript numbers in order of appearance
- Example: `Previous research has shown this.¹`
- Multiple references: `¹,³,⁵` or `¹⁻³` for consecutive

### Reference List Format

**Journal Article (≤4 authors):** List all authors
```
1. Smith AB, Jones CD, Wilson EF. Title of article. Abbrev Journal Name. Year;Volume(Issue):Page range. doi:xxxx
```

**Journal Article (>4 authors):** First 3 authors + "et al."
```
2. Smith AB, Jones CD, Wilson EF, et al. Title of article. Abbrev Journal Name. Year;Volume(Issue):Page range. doi:xxxx
```

**Example:**
```
1. Paine NJ, Horwath JC, Lawson R. Food choice and nutrition in New Zealand. N Z Med J. 2023;136(1578):45-52. doi:10.1.nzj
```

### Journal Abbreviations
Use **Index Medicus** abbreviations:
- New Zealand Medical Journal → N Z Med J
- New England Journal of Medicine → N Engl J Med
- The Lancet → Lancet
- JAMA → JAMA
- BMJ → BMJ

## Figure Requirements

### Technical Specifications

| Format | Resolution | Color Mode | Use |
|--------|------------|------------|-----|
| **TIFF** | 300 dpi minimum | RGB or CMYK | Photographs, clinical images |
| **EPS** | Vector | RGB | Line art, graphs |
| **PNG** | 300 dpi minimum | RGB | Graphs, charts (with transparency) |
| **JPEG** | 300 dpi minimum | RGB | Photographs (lossless compression) |

### Size Requirements

- **Single column:** 87 mm width
- **Double column:** 180 mm width
- **Maximum height:** 240 mm
- **Minimum font size in figures:** 8 point

### Figure Legends

- Required for all figures
- Maximum 40 words per legend
- Define all abbreviations
- Explain symbols, arrows, etc.

### Color Figures

- Color figures accepted
- May incur additional charges for print (online typically free)
- Ensure colorblind-friendly palettes
- Provide grayscale version if essential

## Table Requirements

### Format

- Submit as editable text (Word tables, LaTeX, NOT images)
- Each table on separate page
- Numbered consecutively (Table 1, Table 2, etc.)
- Descriptive title above table
- Footnotes below table as needed

### Table Footnotes

Use symbols in this order: *, †, ‡, §, ¶, **, ††, etc.

**Example:**
```
Table 1: Participant characteristics (N=500)

Characteristic          Value
Age, mean (SD)          45.2 (12.3)
Female, n (%)           275 (55)
BMI, mean (SD)          27.8 (4.2)*

*Data available for 485 participants
```

## Supplementary Material

### Acceptable Formats

- Additional tables
- Additional figures
- Methods details
- Data dictionaries
- Statistical code
- Video (rare, case reports)

### Submission

- Submitted separately from main manuscript
- Referenced in main text (e.g., "see Supplementary Appendix")
- Not copy-edited (appears as submitted)
- Counts toward total figure/table limit if extensive

## Manuscript Structure

### Standard IMRAD Format (Original Articles)

1. **Title Page** (separate file)
   - Title (maximum 150 characters)
   - Running title (maximum 50 characters)
   - Author names and affiliations
   - Corresponding author details
   - Word counts
   - Conflicts of interest
   - Funding sources

2. **Abstract** (300 words, structured)

3. **Introduction** (400-600 words)
   - Background and context
   - Research question/aim
   - Significance

4. **Methods** (800-1,200 words)
   - Study design
   - Setting and participants
   - Data collection
   - Statistical analysis
   - Ethics approval

5. **Results** (1,000-1,500 words)
   - Participant flow
   - Primary outcomes
   - Secondary outcomes
   - Subgroup analyses
   - Adverse events (if applicable)

6. **Discussion** (1,000-1,500 words)
   - Principal findings
   - Comparison with literature
   - Strengths and limitations
   - Implications
   - Conclusions

7. **Acknowledgements**

8. **References**

9. **Tables and Figures**

## Ethics and Reporting Requirements

### Ethics Approval

- Required for all human participant research
- Include ethics committee name and approval number
- Example: "Ethics approval was obtained from the New Zealand Health and Disability Ethics Committee (reference 2023/123)"

### Consent

- Required for case reports and identifiable images
- Example: "Written informed consent was obtained from the patient for publication of this case report"

### Clinical Trials

- Must be registered in approved registry (e.g., ANZCTR, ClinicalTrials.gov)
- Include trial registration number in abstract and methods
- Example: "Trial registration: ACTRN1234567890"

### Reporting Guidelines

NZMJ endorses EQUATOR Network guidelines:

| Study Type | Guideline |
|------------|-----------|
| Randomized trials | CONSORT |
| Observational studies | STROBE |
| Systematic reviews | PRISMA |
| Case reports | CARE |
| Qualitative research | COREQ |
| Economic evaluations | CHEERS |

Include completed checklist as supplementary material.

## Submission Process

### Online Submission

1. Create account on NZMJ submission system
2. Complete manuscript details
3. Upload files:
   - Title page (with author details)
   - Main manuscript (blinded for peer review)
   - Abstract
   - Tables and figures
   - Supplementary material
   - Cover letter
4. Suggest reviewers (optional, 2-3 names)
5. Pay APC (if applicable)
6. Submit

### Cover Letter Requirements

- Manuscript title
- Manuscript type
- Corresponding author details
- Statement of originality
- Conflicts of interest
- Suggested reviewers (optional)
- Any special considerations

### Peer Review

- **Initial screening:** 1-2 weeks
- **Peer review:** 4-6 weeks
- **Decision:** 6-8 weeks total
- **Revision deadline:** Typically 30-60 days

### Publication Fees (APC)

⚠️ **VERIFY CURRENT RATES**

| Article Type | APC (NZD) |
|--------------|-----------|
| Original Article | $2,000-3,000 |
| Review Article | $2,000-3,000 |
| Short Communication | $1,000-1,500 |
| Case Report | $500-1,000 |
| Editorial/Letter | No fee |

**Waivers:** Available for low-income countries, invited editorials

## Post-Acceptance

### Proofreading

- Authors receive proofs for review
- 48-hour turnaround typical
- Only essential corrections allowed

### Open Access

- Optional open access (additional fee)
- CC-BY-NC-ND license typical
- Self-archiving policy: Check SHERPA/RoMEO

### Press and Media

- NZMJ may issue press release for significant findings
- Embargoed press materials coordinated with journal
- Authors should coordinate media inquiries with NZMJ
