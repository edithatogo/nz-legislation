# Channel Matrix

This matrix is the umbrella track's single list of distribution, submission, and
integration surfaces. All rows are preparation-only until the required gates
pass.
Package and listing claims must stay NZ-runtime-only until Australian provider
gates pass. ANZ aliases are transitional compatibility names; this PR does not
rename, publish, mirror, or submit any package or listing.
